# SERMST Web

Aplicação web principal da SERMST, construída com Next.js, React e TypeScript.

## Desenvolvimento

Instalar dependências:

```bash
npm install
```

Rodar em desenvolvimento:

```bash
npm run dev
```

Validar antes de publicar:

```bash
npm run test
npm run lint
npm run build
```
