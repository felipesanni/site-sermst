## Insights críticos do GSC

**1. O blockbuster do site é uma página com intent errado.** `/exames/funcao-encarregado-responsabilidades-e-perfil-ideal/` recebe 125k impressões em 16 meses (28% das impressões totais!) e converte só 346 cliques (CTR 0,3%). É busca de pessoa querendo entender o cargo, não dono querendo contratar SST. **Decisão**: manter a página no novo projeto mas adicionar bloco lateral/inferior com CTA voltado para o gestor que está pesquisando ("Você gerencia encarregados? Sua empresa cumpre o PCMSO da equipe? Falar com a SERMST"). Convertemos parte do tráfego sem perder.

**2. O CTR médio de 0,4% é catastrófico.** A média de mercado para SERPs de SST é 2-5%. O problema é title/description genéricos do WordPress. **Decisão**: no novo projeto, reescrever TODOS os titles/descriptions com fórmula consistente (benefício + cidade + número/ano).

**3. Existem 3 páginas geo-bairro confirmadas, não 2:**
- `/sao-paulo/pinheiros/clinica-para-exame-admissional-em-pinheiros/` — 73 cliques
- `/sao-paulo/onde-realizar-exame-admissional-na-liberdade-sao-paulo/` — 30 cliques
- `/sao-paulo/exame-admissional-na-vila-mariana/` — 17 cliques (NOVA)

Total: 120 cliques/16 meses só de bairro. Não é desprezível, mas também não justifica criar matriz `/[bairro]/` no novo projeto. **Decisão recomendada**: 301 das 3 páginas para `/servicos/exame-admissional-expresso/sao-paulo` no novo, perde-se ~100 cliques/16 meses (~6/mês). Aceitável.

**4. Canibalização SEO real:** existem 2 URLs para o mesmo conteúdo NR-1: `/medicina-do-trabalho/nr-1-atualizada/` (45 cliques) e `/uncategorized/nr-1-atualizada/` (5 cliques). Site atual tem essa duplicação acumulando autoridade dividida. **Decisão**: 301 de uncategorized → medicina-do-trabalho.
